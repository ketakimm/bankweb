# 💰 Bank Web App

A simple web application to calculate interest and generate a downloadable report.

## 🚀 Features
- User login system
- Interest calculation (Savings, FD, Loan)
- Year-wise breakdown table
- PDF report generation
- Clean multi-page navigation

## 🛠️ Tech Used
- HTML
- CSS
- JavaScript

## 📂 Pages
- Login Page
- Dashboard
- Thank You Page
